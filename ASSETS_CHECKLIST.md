# Portfolio Assets Checklist

Please gather the following materials and place them in the corresponding folders I have created in `c:\PortFolio-2d\assets\`. These will populate the interactive modals in your DevTown portfolio.

## 📁 `assets/profile/`
- [ ] `headshot.png` or `headshot.jpg` (A professional or high-quality picture of you).
- [ ] Your Bio/About Me text (You can just provide this in the chat).

## 📁 `assets/resume/`
- [ ] `resume.pdf` (Your most up-to-date resume).

## 📁 `assets/projects/`
For your top 3-4 projects, please gather:
- [ ] 1-2 high-quality screenshots per project (e.g., `project1_screenshot1.png`).
- [ ] (Optional) Short GIF of the project in action.
- [ ] In the chat, please provide:
  - Project Title
  - Short Description & Tech Stack
  - GitHub Repo URL & Live Demo URL
  - (Optional) YouTube/Loom video link if you have a video demo.

## 📁 `assets/certificates/`
- [ ] Images/PDFs of any certificates or hackathon badges you want to display (e.g., `aws_cert.png`, `hackathon_winner.jpg`).

## 💬 Provide in Chat (No files needed)
- **Social Links:** Your GitHub, LinkedIn, Twitter, and Email address for the Contact Hub.
- **Skills List:** A list of your technical skills (e.g., React, Python, Node.js, AWS). I will automatically fetch the official logos for these using a developer icon library, so you don't need to download icons!

Once you've placed the files in their respective folders and provided the text details in the chat, we will be ready to initialize the project and start building!
